# Disconnect-TdService

## SYNOPSIS
Disconnects you from the TOPdesk service and invalidates your login token.

## SYNTAX

```
Disconnect-TdService [<CommonParameters>]
```

## DESCRIPTION
Disconnects you from the TOPdesk service and invalidates your login token.

## EXAMPLES
